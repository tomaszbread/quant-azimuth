QuantAzimuth


	
## Setup

To run this project, clone repository 

```
git clone https://github.com/tomaszbread/quant-azimuth.git
```

Npm install in server folder 

Npm install in root project folder 

To run server:

```
npm run server 
```

To run project: 

```
npm run start
```
